# Notification Generator

O **Notification Generator** é uma ferramenta que automatiza a criação de sistemas de notificação para projetos de software, permitindo gerar rapidamente estruturas para envio de e-mails, SMS, push notifications e mensagens em tempo real para aplicações em .NET, Node.js, Python, Go, entre outras tecnologias.

## Tecnologia

- **Linguagem principal:** Node.js  
- **Por que da escolha:** Excelente para aplicações em tempo real, push notifications, WebSockets e escalável para múltiplos canais de envio de mensagens.
